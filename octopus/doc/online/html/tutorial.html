Using Octopus in Six Steps

Working with Octopus, you must perform the following steps:

1. Create an Octopus project, or - if you want to use the code generation - create a Java project and add the Octopus nature to it.
 2. Create the files that constitute the UML model.
 3. Create the files that contain the OCL expressions.
 4. Check the OCL expressions. If there are errors, remove them and recheck.
 5. View the OCL expressions in the Business Modeling Syntax (optionally).
 6. Generate Java code for the model and expressions (optionally). Note that this feature is still a beta version.
