The AST View

The AST View is a window showing the internal representation of an OCL expression (the Abstract Syntax Tree) in a tree hierarchy. It can be opened by selecting a UML element in the Model View, and selecting 'Open AST Viewer' from the context menu.

The Abstract Syntax Tree is shown in an XML fashion. Note however, that this is not compliant to the XMI defined in the OCL standard.

(The implementation of this view uses jdom, an open source, tree-based, pure Java API for parsing, creating, manipulating, and serializing XML documents. More info can be found at www.jdom.org.)
