The Type Hierarchy View

The Type Hierarchy View is a window showing the sub and super types of a UML element in a tree hierarchy. It can be opened by selecting a UML element in the Model View, and selecting 'Open Type Hierarchy' from the context menu.

Because a tree element can have at most one parent, not all supertypes of a class can be shown. The first that is encountered is shown. Note that interfaces are shown only when the selected UML element itself is an interface.
