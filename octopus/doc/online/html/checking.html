Checking OCL Expressions

The expressions in the OCL files will be checked whenever the Octopus project is rebuild. You can make sure the project is rebuild by using the menu options 'Project>Rebuild Project' or 'Project>Rebuild All'. Saving the OCL file will also cause a rebuild of the project, if the option 'Perform build automatically on resource modification' is turned on. You can find this option under 'Window>Preferences>Workbench'.

The first step in the build process is the checking of the UML files that constitute the base of the model. If errors are found, and therefore the base model cannot be build, no OCL files are checked. If the base model is correct, the OCL files are checked. Errors in the OCL files are reported in the 'Tasks' window.
