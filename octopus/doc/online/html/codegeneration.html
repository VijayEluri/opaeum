Generating Java Code

Octopus is able to create Java code from the UML model . Optionally code implementing the OCL expressions is generated as well. Java code can only be generated for Java projects that have the Octopus nature (see An Octopus Project).

To generate code open the `Octopus' context menu on the Java project and select `Generate Java Code', or use the button with the `J' and the pencil on it. Another way to start code generation is to select the `Generate Java Code' entry in the Octopus menu.

Code Generation Options

You may customize the code generation process by selecting or deselecting the options given in the project properties for Octopus Code Generation (see Code Generation Properties).

An explanation on how to use the generated code can be found elsewhere.

Code Generation Example

On the Klasse Objecten website you can find an code generation example called dvdshop. When you upzip the file the following folders will be created:
  • octopus-model-dvdshop: contains the model and expression folder for the Octopus project.
  • poseidon-model-dvdshop: contains the dvdshop model in Poseidon format and in XMI format generated by Poseidon.
  • xml-store-for-dvdshop: contains the stored instances created with an application generated from the Octopus dvdshop project.
In order for you to recreate the dvdshop aplication you have to perform the following:
 1. Start Octopus and create a Java project called dvdshop or something you find more appropriate.
 2. Add the Octopus nature to this project. Use the Add Octopus Nature button or the entry in the Octopus menu or context menu.
 3. Import the XMI file from the poseidon-model-dvdshop folder into the project. Use the XMI import button or the XMI import entry in the Octopus menu.
 4. Remove the package called java from the model folder. (Poseidon always creates this package when using the basic types Integer, String, etc.)
 5. Import the contents of the expression folder from octopus-model-dvdshop. Use File>Import>File System.
 6. Open the project properties using the context menu. Select Octopus Code Generation. Check the following options, uncheck all other options.
 · Generate code for OCL expressions
 · Generate user interface prototype
 · Generate XML storage facility
 7. Generate Java code for the project. Use the Generate Java Code button or the entry in the Octopus menu or context menu.
 8. Edit the file called dvdshop.xml in the xml-store-for-dvdshop folder. Change the entry

xsi:noNamespaceSchemaLocation="file:/d:/octopus/workspace/octopuscodegen/octopus-storage.xsd

into

xsi:noNamespaceSchemaLocation="file:$ECLIPSE-HOME/plugins/nl.klasse.octopus.codegen_1.0.0/octopus-storage.xsd

$ECLIPSE-HOME should be replaced by the location on your system where Eclipse is installed, e.g. d:/javatools/eclipse2.1.1.

9. Create a run for the generated application. See The Generated User Interface for the details. Launch the application.
 10. In the running dvdshop application select the menu entry File>Open. In the file selection window select the file called dvdshop.xml in the xml-store-for-dvdshop folder. Press the Refresh button or select another tab in the dvdshop application.
