Creating OCL Expressions

The main function of Octopus is to create and check OCL expressions. OCL expressions are expected to be written in files with the extension '.ocl'. Each OCL file needs to be structured according to the rules in the OCL specification version 2.0. All OCL files should be placed under the dedicated expression folder in the Octopus project. Unlike the model folder, you are free to decide about the folder structure inside the expression folder.

Octopus provides a dedicated OCL editor, which offers syntax highlighting and (a rough form of) code completion. You can open the editor in the standard Eclipse ways on any file with extension '.ocl'.

Examples of OCL files can be found in the example projects.
