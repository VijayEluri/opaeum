Octopus

Octopus is a software modeling tool, created by Klasse Objecten, that provides support for working with OCL expressions in a UML model. Working with OCL expressions gives you the possiblity to express more information on the model than can be expressed in the diagrams that constitute the UML model.

Since January 1, 2006 octopus is an open source project hosted at sourceforge.
